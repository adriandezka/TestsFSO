# Simulador d'Exàmens Tipus Test - Fonaments de Sistemes Operatius (FSO) - Universitat Rovira i Virgili (URV)
Programa en Java que simula un dels examens tipus test de l'assignatura Fonaments de Sistemes Operatius del Grau en Enginyeria Informàtica de la URV, amb preguntes reals d’anys anteriors, control de temps idèntic al de les proves oficials i càlcul de la nota final aplicant la fórmula d’avaluació (cada resposta incorrecta resta punts).

Aquest repositori té la finalitat de servir per a pràcticar per als examens de tipus test de l'assignatura d'FSO. La seva finalitat és compartir la manera d'estudi que jo he utilitzat durant el passat curs. Aquest és el meu primer repositori amb finalitats públiques, així que si observes cap problema o vols realitzar cap aportació, recomano contactarme directament. Gràcies!

## Execució
Compila els fitxers:
```bash
javac *.java
java Launcher
```
